# NAME

Sisimai - It's a core module of bounceHammer version 4

# SYNOPSIS

    use Sisimai;

# DESCRIPTION

Sisimai is a core module of bounceHammer version. 3, is a Perl module for 
analyzing email bounce. "Sisimai" stands for SISI "Mail Analyzing Interface".
